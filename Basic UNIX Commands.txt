Basic UNIX Commands
1. pwd ‚Äì Print Working Directory

Explanation: Shows the current directory you are in.

Syntax:

pwd


Application: To check your current location in the filesystem.

Output:

/home/kathir/Documents

2. ls ‚Äì List Directory Contents

Explanation: Lists files and directories in the current location.

Syntax:

ls [options] [directory]


Options:

ls -l ‚Üí detailed list with permissions, size, time.

ls -a ‚Üí shows hidden files (starting with .).

Application: To view contents of directories.

Output:

file1.txt   file2.c   myfolder   .hiddenfile

3. cd ‚Äì Change Directory

Explanation: Used to navigate between directories.

Syntax:

cd [directory]


Application: To move to another directory.

Output:

cd /home/kathir/Desktop

4. cat ‚Äì Concatenate & Display File

Explanation: Displays contents of a file.

Syntax:

cat filename


Application: To quickly read small text files.

Output:

$ cat hello.txt
Hello, UNIX World!

5. touch ‚Äì Create Empty File

Explanation: Creates a new empty file or updates timestamp of an existing file.

Syntax:

touch filename


Application: To make new empty files.

Output:

$ touch newfile.txt
$ ls
newfile.txt

6. mkdir ‚Äì Make Directory

Explanation: Creates a new directory.

Syntax:

mkdir directory_name


Application: To organize files into folders.

Output:

$ mkdir projects
$ ls
projects

7. rmdir ‚Äì Remove Directory

Explanation: Deletes an empty directory.

Syntax:

rmdir directory_name


Application: To remove unused directories.

Output:

$ rmdir projects

8. rm ‚Äì Remove Files/Directories

Explanation: Deletes files or directories (with -r).

Syntax:

rm filename
rm -r directory


Application: To delete files.

Output:

$ rm old.txt

9. cp ‚Äì Copy Files

Explanation: Copies files or directories.

Syntax:

cp source destination
cp -r dir1 dir2


Application: To duplicate files or directories.

Output:

$ cp hello.txt backup.txt

10. mv ‚Äì Move or Rename

Explanation: Moves or renames files.

Syntax:

mv oldname newname
mv file /path/to/directory


Application: To rename or organize files.

Output:

$ mv hello.txt greetings.txt

11. man ‚Äì Manual Pages

Explanation: Displays help and documentation for commands.

Syntax:

man command


Application: To understand how a command works.

Output:

$ man ls

12. echo ‚Äì Print Message

Explanation: Prints text/variables to terminal.

Syntax:

echo "message"


Application: To test shell scripts or print variables.

Output:

$ echo "Hello UNIX"
Hello UNIX

13. whoami ‚Äì Current User

Explanation: Shows the current logged-in user.

Syntax:

whoami


Application: To know which user is active.

Output:

kathir

14. date ‚Äì Display Date & Time

Explanation: Shows current system date and time.

Syntax:

date


Application: To check date & time.

Output:

Fri Sep 5 21:15:42 IST 2025

15. cal ‚Äì Display Calendar

Explanation: Shows calendar of current or given month/year.

Syntax:

cal [month] [year]


Application: To check calendar.

Output:

$ cal 9 2025
